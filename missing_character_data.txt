Missing Character Data from Mausritter SRD 2.3
==============================================

This file lists character-related data from the SRD that is NOT currently
included in the character generator, or where the implementation differs
from the official rules.

Last updated: 2026-01-12

RULES DIFFERENCES
-----------------
1. Attribute Swap (NOT IMPLEMENTED)
   - SRD: "You may then swap any two attributes"
   - Generator: Does not offer this option
   - This is a player choice that could be offered via the HTML interface

MISSING WEAPON DATA
-------------------
1. Ranged Weapons (not in weapon selection)
   - Light ranged: d6 damage, main paw (Sling, light crossbow) - Cost: 10p
   - Heavy ranged: d8 damage, both paws (Bow, heavy crossbow) - Cost: 40p

2. Ammunition (not tracked)
   - Stones, pouch: Carried in body slot - Cost: 1p
   - Arrows, quiver: Carried in body slot - Cost: 5p

3. Improvised Weapons (not in selection)
   - d6 damage, wielded in main paw or both paws
   - Special: Always mark usage after a fight

SPELL DETAILS (NOT IMPLEMENTED)
-------------------------------
Some backgrounds provide spells, but spell mechanics are not included:
- Spells provided: Magic Missile, Be Understood, Heal, Restore, Darkness
- Not implemented: Spell casting rules, spell dice, mishaps, dooms
- Spell recharge mechanics not shown

HIRELING DETAILS (PARTIALLY REPRESENTED)
----------------------------------------
Some backgrounds provide hirelings (shown as text only):
- Loyal beetle (Beetleherd background)
- Drunken torchbearer (Ale brewer background)
- Pack rat (Merchant background)

Hireling stats, loyalty, and rules from SRD "Hirelings" section not included.

CONDITIONS SYSTEM (NOT IMPLEMENTED)
-----------------------------------
Conditions occupy inventory slots and have specific effects/clear requirements:

| Condition   | Effect                           | Clear                      |
|-------------|----------------------------------|----------------------------|
| Exhausted   | No running, disadvantage on DEX  | Long rest                  |
| Frightened  | Disadvantage on WIL saves        | Safe rest                  |
| Hungry      | No HP recovery from rest         | Eat a meal                 |
| Injured     | Max HP reduced                   | Full rest + medical aid    |
| Poisoned    | Disadvantage on all saves        | Antidote or time           |

IMPLEMENTED FEATURES
--------------------
The following features ARE now implemented:
- Attribute generation: Roll 3d6, keep two highest (range 2-12 per SRD 2.3)
- Structured inventory (Paw slots, Body slots, Pack slots)
- Max/Current tracking for attributes and HP
- Level, XP, Grit fields
- Pips with max fixed at 250 (purse capacity)
- Birthsign with disposition
- Coat (color + pattern combined)
- Physical detail (d66 table)
- Background items placed appropriately (armor to body, others to pack)
- Generate New Character with custom dialogs (Accept/Regenerate flow)
- Current values enforced to not exceed max values
- Weapon selection: User prompted to "Select weapon" rather than random assignment
- Item selector with search functionality and categorized items
- Item usage markers (3 dots per slot, click to mark usage)
- Torches/lanterns: 6-use system (each marker can be half-filled for 6 total uses)
- Depleted items shown with strikethrough when all markers used
- Two-slot items supported (heavy weapons, heavy armor)
- Empty slots have disabled usage markers
- Purse item added to Miscellaneous (holds 250 pips)
- Dice roller with custom notation support

NPC-SPECIFIC DATA (for GMs, not player characters)
--------------------------------------------------
These tables are for NPCs, not player character generation:

1. NPC Appearance (d20 table)
   - Soulful eyes, Bright patched clothes, Wreath of daisies, etc.

2. NPC Quirk (d20 table)
   - Constantly grooming, Obsessed with weather, Very high energy, etc.

3. NPC Wants (d20 table)
   - Freedom, Safety, Escape, Excitement, Power, etc.

4. NPC Relationship (d20 table)
   - Parent, Sibling, Cousin, Married, Former lovers, etc.

5. NPC Social Position / Starting Wealth
   - Peasant: d6p
   - Tradesmouse: d6 x 10p
   - Guildmouse: d4 x 100p
   - Noblemouse: d4 x 1000p
